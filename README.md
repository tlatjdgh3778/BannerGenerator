# BannerGenerator

React를 이용해서 만드는 Banner 생성 웹앱

* React
* Material UI + Styled-Component
  * Material UI 사용 연습
* Jest 로 테스트 코드 작성 연습
* Figma로 프로토타이핑



## 프로토타이핑
프로젝트를 만들기 전에 먼저 **Figma** 를 사용해서 전체적인 프로토타이핑을 하였다. **Figma**를 사용하는 것이 이번이 처음이라 간단하게만 만들었다.
